# Frontend (Next.js)

Next.js dashboard for visual sentiment analysis and market prediction.

## Setup

```bash
pnpm install  # or npm install
pnpm dev
```

The app will run on `http://localhost:3000` by default.

## Features

- Dashboard with statistics and recent analyses
- Image upload and sentiment analysis
- Real-time data refresh
- Day theme with modern UI

## Environment Variables

Set `NEXT_PUBLIC_API_BASE` if your backend is not running on `http://localhost:5000`.
